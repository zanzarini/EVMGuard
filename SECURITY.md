# Security Policy

## Reporting a vulnerability

Do not disclose security vulnerabilities through public issues. Use a private security advisory in this repository and include reproduction steps, affected versions, impact, and suggested mitigations where possible.

## Scope

Security reports are relevant to vulnerabilities in EVMGuard source code, release artifacts, CI workflows, or documented deployment instructions.

## Guarantees

EVMGuard is an analysis tool. It does not guarantee that an analyzed transaction or contract is safe.
